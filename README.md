# Bike-service-Application

<h4>Steps to run</h4>

1) Clone the repository
```
git clone https://github.com/vino583/Bike-service-Application
```

2) Change directory

```
cd client
```

3) Install dependencies

```
npm install 
```

4) Start frontend server

```
npm start
```
